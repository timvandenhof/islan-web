Rocket League

Rocket League is een racevoetbalspel ontwikkeld door Psyonix en wordt omschreven als voetbal met raket aangedreven auto's.

Speel met tot 8 spelers verdeeld over 2 teams in de Rocket League arena. Zweef hoog door de lucht en voer gedurfde acrobatische capriolen uit en agressieve tactieken om je rivaliserende spelers met supersonische snelheden neer te halen.

Deze game is Free to Play en ondersteunt crossplay op Windows, Xbox, Linux, MacOS, Playstation en Nintendo Switch.

Minecraft

Bedrock Edition / Java Edition

Klassieke sandbox game waarin je je uren kan vermaken in een oneindige wereld.

Deze game is betaald en ondersteunt crossplay op Windows, Xbox, Android, iOS en via een workaround ook op PS4 en Switch. Daarnaast is ook de Java edition beschikbaar, de werelden tussen beide zijn niet gedeeld.

Game mode is vanilla: normal survival, zonder PvP.
